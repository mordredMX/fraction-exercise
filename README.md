# fraction-exercise

In order to build the program
- java 8
- maven 3
execute:
mvn clean install

In order run the program:
- Go to target folder
- execute: java -jar fractions-1.0.0-SNAPSHOT.jar

Examples of valid operations:
- -4 + 1
- 4_1/5 / -1/5
- 4 + 2
- 31/5 * 5
- 0 * 0
- 1_4/5 + 5_6/8
- 2 - -3_1/5
